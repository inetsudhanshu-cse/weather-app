# 🌤 Weather App

A modern and responsive weather application that provides real-time weather updates using the OpenWeather API. Users can search for any city and get detailed weather information including temperature, humidity, and wind speed.

---

## 🚀 Live Demo
👉 https://inetsudhanshu-cse.github.io/weather-app/

---

## 📌 Features

- 🌍 Search weather by city name
- 🌡 Real-time temperature display
- 💧 Humidity information
- 🌬 Wind speed details
- 🎨 Modern glassmorphism UI
- 📱 Responsive design
- 🖼 Dynamic weather icons based on conditions

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Glassmorphism UI)
- JavaScript (Vanilla JS)
- OpenWeather API

---

## ⚙️ How It Works

1. User enters a city name
2. App sends request to OpenWeather API
3. API returns real-time weather data
4. Data is displayed dynamically on UI
